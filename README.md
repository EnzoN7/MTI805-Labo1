# Utilisation du programme

La détection des mains est toujours activée. 

## Mode 'détection faciale'

* Run **python cli.py -d**

## Mode 'reconnaissance faciale'

* Run **python cli.py -r**

## Besoin d'aide ?

* Run **python cli.py -h**

Pour fermer la fenêtre, appuyer sur une touche du clavier.
